```remove_public_notification ```
====
Este modulo elimina de los remitentes de un email de tipo notificacion, 
aquellos usuarios que esten incluidos en el grupo public.
Este modulo es utilizado en website_sale cuando se configura todo el sistema
con el usuario public y no se desean notificaciones en los cambios de estado
 de algunos objetos como el pedido de venta, etc.
