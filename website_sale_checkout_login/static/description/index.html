Añade una página para inicio de sesión o registro de usuario al proceso de compra de la tienda online.

Dicho registro puede crear el usuario a partir de una Plantilla de usuario, por lo que se recomienda realizar la siguiente configuración.

Configuración

En la página "Ventas > Clientes" estableceremos los siguientes valores para "Template User":

  • ¿Es una empresa?: Si
  • Pestaña "Ventas & Compras" > Cliente: Si

Nota: Si hubiese que modificar alguna propiedad del usuario (que no del cliente "Tempate user") lo haríamos desde la página "Configuración > Usuarios > Usuarios" filtrando en el listado por "Usuarios no activos" para que se muestre.

Nota: En implantaciones multicompañía, "Template User" (tanto el cliente como el usuario web) ha de pertenecer a la compañía del sitio web.

En la página "Configuración > Configuraciones Generales" estableceremos los siguientes valores:

  • Habilitar restablecimiento de la contraseña desde la página de inicio de sesión: Si
  • Permitir ingresar a usuarios externos: Si
  • Plantilla de usuario para los nuevos usuarios creados a través del ingreso: Template User
